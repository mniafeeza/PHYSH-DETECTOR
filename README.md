
To access and add the extension to Chrome, follow these steps:

Accessing the Extension Folder:

obtain the folder containing the extension files (HTML, CSS, JavaScript, and any other necessary assets).
The folder should contain all the files required for the extension to function properly.

Adding the Extension to Chrome:

Open Google Chrome on your computer.
Click on the three vertical dots in the top right corner to open the Chrome menu.
From the menu, select "More tools" and then "Extensions." Alternatively, you can type "chrome://extensions/" in the address bar and press Enter.
In the Extensions tab, ensure that the "Developer mode" toggle switch in the top right corner is turned on.
Click on the "Load unpacked" button.
Navigate to the folder containing the extension files and select the folder.
Click "Select Folder" to add the extension to Chrome.
The extension should now appear in the list of installed extensions in Chrome.
User Guide:

Accessing the Extension:

Once the extension is installed, you can access it by clicking on the puzzle piece icon (Extensions) next to the address bar in Chrome.
Locate the extension icon among the installed extensions and click on it to open the extension.
Using the Extension:

Upon opening the extension, you will see the interface, which may include a popup text box or other elements depending on the design.
Follow the instructions provided in the extension interface to interact with it.
Input the email content or any other relevant information as instructed.
Follow the prompts to receive feedback on the legitimacy of the email or perform any other actions supported by the extension.
